# **Чеклист**
Тестовое задание

### Описание
------------
Верстка секции с прогресс-баром и кнопкой "Открыть чеклист"

### В проекте были использованы следующие технологии:
-----------------------------------------------------
* Методология БЭМ
* Организация файловой структуры Nested
* Flexbox верстка
